# coolsculpting brazil static site

## Commands
- `npm i` - installs dependencies
- `yarn start` - starts local development
- `npm run build`, `yarn build` - builds the entire site.

## Environments
Each branch is tied to an environment
- `develop` - https://develop--pdcoe-static-coolsculpting-br.netlify.app/
- `main` - https://www.coolsculpting.com.br/
